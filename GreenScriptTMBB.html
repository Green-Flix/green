Instruções do Green Script TMBB

Para começar a usar o programa Green Script TMBB, siga estas instruções:

  1. Execute o programa GreenScriptTMBB.exe.
  2. Quando solicitado, insira a sua chave da API do TMDb.
  3. Escolha o diretório onde deseja salvar os arquivos XML. Se não especificado, será usado o diretório padrão.

Para editar o diretório de salvamento, siga estas etapas:

  1. Navegue até o diretório onde o programa está instalado.
  2. Abra o arquivo "save_dir.txt" com um editor de texto (como o Bloco de Notas).
  3. Substitua o caminho do diretório existente pelo seu novo diretório de salvamento.
  4. Salve o arquivo.

Se precisar editar a chave da API posteriormente, siga estas etapas:

  1. Navegue até o diretório onde o programa está instalado.
  2. Abra o arquivo "api_key.txt" com um editor de texto (como o Bloco de Notas).
  3. Substitua o conteúdo existente pelo seu nova chave da API.
  4. Salve o arquivo.

Lembre-se de manter sua chave da API confidencial e não compartilhá-la com outras pessoas.
